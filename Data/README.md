# Data

All source datasets used in this study are publicly accessible through Google Earth Engine or official institutional repositories, including Landsat surface reflectance and thermal imagery, the 1-km station-based gridded precipitation and potential evapotranspiration dataset for the Qinghai-Tibetan Plateau from the National Tibetan Plateau Data Center, MERIT DEM, the CSIC global SPEI database, TerraClimate, and GLDAS-2.

The field soil-moisture validation dataset is provided as `Field_soil_moisture_validation_Aug2024.xlsx` via Baidu Netdisk:

- Download link: https://pan.baidu.com/s/17i7qwsl9endxgbK5FfF6Rw?pwd=2tuk
- Extraction code: `2tuk`
