# MS-TCFNet

This repository accompanies the manuscript `A scale-aware and interpretable deep-learning framework for 30-m ecohydrological drought reconstruction in alpine peatlands`.

MS-TCFNet is a multi-scale supervised spatiotemporal deep-learning network developed to reconstruct 30-m Ecohydrological Drought Index (EHDI) fields from monthly predictor sequences under coarse meteorological supervision.

## Repository structure

- [`Code/`](Code/): source code and model-architecture documentation for MS-TCFNet.
- [`Data/`](Data/): source-dataset access information and the download route for the field soil-moisture validation dataset.

## Availability

The source code used in this study is publicly available in the [`Code/`](Code/) directory. Information on the public source datasets and access to the field soil-moisture validation data is provided in the [`Data/`](Data/) directory.
